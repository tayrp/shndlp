ShnDLP GUI Wrapper

ShnDLP is a lightweight graphical desktop application built on top of yt-dlp. It runs quietly in the background and automatically downloads supported videos when you copy a link to your clipboard.

Requirement
yt-dlp must be installed and available in your system PATH.

Verify installation: 
yt-dlp --version

How it works

  • Launch the ShnDLP GUI application
  • Default configuration is enabled automatically
  • Copy a supported video or playlist URL
  • ShnDLP detects the link from your clipboard
  • The download begins automatically using yt-dlp

Features

  • Automatic clipboard monitoring
  • No command line usage required
  • Supports all sites handled by yt-dlp
  • Background downloads
  • Cross-platform (Windows, macOS, Linux)

Installation

1. Install yt-dlp

Install yt-dlp using your preferred method and ensure it is on PATH: 

pip install -U yt-dlp

2. Run ShnDLP

  • Extract the downloaded archive
  • Launch the ShnDLP application
  • Leave it running in the background

3. Copy a link

Copy any supported video or playlist URL. ShnDLP will automatically start downloading it.

Configuration

ShnDLP includes a default configuration on first launch. Advanced users may adjust:

  • Download location
  • Clipboard monitoring behavior

Notes

  • This application runs locally on your machine
  • No web service or server component
  • Follow applicable laws and site terms of service
