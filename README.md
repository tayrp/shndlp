# ShnDLP

Get latest versions on the releases tab.

Requirements:
Install yt-dlp either with python or manually. (Preferably use Python)
[yt-dlp](https://github.com/yt-dlp/yt-dlp/releases/latest)
[ffmpeg](https://www.ffmpeg.org/download.html)

ShnDLP can only download audio/video files, as it is just a 'wrapper' for [yt-dlp](https://github.com/yt-dlp/yt-dlp/releases/latest)

ShnDLP does not receive or collect any data, the program only monitors the clipboard to automatically download videos.
ShnDLP will never receive or collect any user data.
